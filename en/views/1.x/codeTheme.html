    Multiple code themes

    reco_luan 7/5/2020

    注意

    自 1.6.10 废弃。

    # Theme style

    Go to prismjs (opens new window).

    # Set the theme

    // .vuepress/config.js

module.exports = {
  theme: 'reco',
  themeConfig: {
    /**
     * support for
     * 'default'
     * 'funky'
     * 'okaidia'
     * 'solarizedlight'
     * 'tomorrow'
     */
    codeTheme: 'tomorrow' // default 'tomorrow'
  }
}
